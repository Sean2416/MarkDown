       - `GCIPiece = 0 ` - 未入倉

     - 放行判斷

       - `ReleaseTime = null & GCIPiece > 0 `
         - `CancelRelease = null` - 未放行
         - 取消 
       - `ReleaseTime != null`
         - 放行
       - 其他 
         - 未放行 或欠稅

     - 進倉判斷

       - `GCIPiece = 0`
         - `CloseMark = LK ` - 已鎖定
         - `CloseMark = PP` - 已列印未進倉證明
         - 未進倉
       - ` GCIPiece <> Piece ` 
         - 部份進倉
       - `GCIPiece = Piece & CloseMark <> *` 
         - 完全進倉

     - 出倉判斷

       - `GCOPiece = 0`
         - 未出倉
       - `GCOPiece <> Piece`
         - 部份出倉
       - `CloseMark`
         - `= *E ` - 退關
         - `= *I ` - 退運
         - `= *M` - 移倉
       - 全部出艙
